# EcoWise Project

EcoWise is a sustainable waste management and recycling assistant. It helps users identify recyclable items using AI and find nearby recycling centers.

## Features
- **AI-Powered Item Analysis**: Upload photos of waste items to identify them and get recycling advice.
- **Recycling Analytics**: Track your recycling habits and environmental impact.
- **Center Locator**: Find nearby recycling centers with directions.
- **Leaderboard**: Compete with others to maximize your positive impact.

## Project Structure
- `frontend/`: Contains the web interface (HTML, CSS, JS).
- `backend/`: Contains the server-side logic and AI models.
